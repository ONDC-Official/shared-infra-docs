# Lookup API — Troubleshooting Guide

> **Last updated:** 2026-04-14

## About This Document

This document provides troubleshooting guidance for the ONDC Registry Lookup API, covering both **v1** and **v2** versions. It is intended to help developers and operators diagnose and resolve issues encountered when calling the lookup endpoint.

Topics covered include common failure scenarios, their likely root causes, and step-by-step resolution approaches across areas such as authentication, request schema validation, participant discovery, and server-side errors. Where behaviour differs between v1 and v2, differences are called out explicitly.
